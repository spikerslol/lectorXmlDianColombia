# lectorXmlDianColombia
Lector de XML DIAN de documentos electronicos COLOMBIA

Proyecto creado en especial para los contadores y auxiliares contables de colombia.
Lee y extrae la informacion de archivos XML de los documentos electronicos validos por la DIAN para faciliar al analisis de datos y la importacion a sistemas contables.
Creado para la comunidad con ayuda de IA
Libre para uso de cualquier persona.
No utilizar para la venta.
